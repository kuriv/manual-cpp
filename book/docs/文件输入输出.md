# 文件输入输出

执行下面的代码，打开指定文件并读取数据。

```cpp
#include <iostream>
#include <fstream>

using namespace std;

int main(int argc, char const *argv[])
{
    ifstream file;
    file.open("index.txt");
    int x, y, z;
    file >> x >> y >> z;
    cout << x << endl;
    cout << y << endl;
    cout << z << endl;
    file.close();
    return 0;
}
```

执行下面的代码，打开指定文件并写入数据。

```cpp
#include <iostream>
#include <fstream>

using namespace std;

int main(int argc, char const *argv[])
{
    ofstream file("index.txt");
    file << 233 << " " << 6.66 << " " << "Hello World!";
    file.close();
    return 0;
}
```
