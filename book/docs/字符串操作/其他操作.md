# 其他操作

执行下面的代码，提取字符串中指定位置的字符。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    string str("Hello World!");
    cout << str.at(0) << endl;
    cout << str.at(4) << endl;
    return 0;
}
```

执行下面的代码，删除字符串指定的部分。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    string str("Hello World!");
    cout << str.erase(2, 3) << endl;
    return 0;
}
```

执行下面的代码，清空字符串。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    string str("Hello World!");
    str.clear();
    cout << str << endl;
    return 0;
}
```

执行下面的代码，检测字符串是否为空。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    string str("Hello World!");
    str.clear();
    cout << str.empty() << endl;
    return 0;
}
```

执行下面的代码，获取字符串的长度。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    string str("Hello World!");
    cout << str.length() << endl;
    cout << str.size() << endl;
    return 0;
}
```

执行下面的代码，获取字符串的存储空间。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    string str("Hello World!");
    cout << str.capacity() << endl;
    str.erase(2, 3);
    cout << str.capacity() << endl;
    return 0;
}
```
