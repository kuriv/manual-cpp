# 字符串搜索

执行下面的代码，再字符串中搜索一个字符或一个子串。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    string str("Hello World!");
    cout << str.find('o') << endl;
    cout << str.find('o', 6) << endl;
    cout << str.find("Wo") << endl;
    cout << str.find("Wo", 7) << endl;
    cout << str.find("WoW") << endl;
    return 0;
}
```
