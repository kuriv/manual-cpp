# 字符串赋值

C++ 提供了几个重载函数，用于赋予字符串新的内容。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    string str1("Hello");
    str1.assign("Welcome");
    cout << str1 << endl;
    string str2("Hello");
    str2.assign("Welcome", 3, 4);
    cout << str2 << endl;
    string str3("Hello");
    str3.assign("Welcome", 2);
    cout << str3 << endl;
    string str4("Hello");
    str4.assign(6, '!');
    cout << str4 << endl;
    return 0;
}
```
