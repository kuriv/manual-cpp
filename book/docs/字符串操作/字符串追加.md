# 字符串追加

C++ 提供了几个重载函数来向一个字符串追加新的内容。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    string str1("Hello");
    str1.append(" World!");
    cout << str1 << endl;
    string str2("Hello");
    str2.append(" World!", 3);
    cout << str2 << endl;
    string str3("Hello");
    str3.append(" World!", 5, 2);
    cout << str3 << endl;
    string str4("Hello");
    str4.append(6, '!');
    cout << str4 << endl;
    return 0;
}
```
