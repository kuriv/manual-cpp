# 标准输入输出

执行下面的代码，从键盘读取输入并输出。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    int input;
    cin >> input;
    cout << input << endl;
    return 0;
}
```

也可以用一条语句读取多个输入。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    int input1, input2, input3;
    cin >> input1 >> input2 >> input3;
    cout << "input1: " << input1 << "." << endl;
    cout << "input2: " << input2 << "." << endl;
    cout << "input3: " << input3 << "." << endl;
    return 0;
}
```
