# 引用类型

可以通过引用变量来访问和修改存储在变量中的原数据。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    int num = 1;
    int &r = num;
    cout << num << endl;
    cout << r << endl;
    num++;
    cout << num << endl;
    cout << r << endl;
    num = 10;
    cout << num << endl;
    cout << r << endl;
    return 0;
}
```
