# 读取输入

执行下面的代码，从键盘读取输入。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    int x;
    cin >> x;
    cout << x << endl;
    return 0;
}
```

也可以用一条语句读取多个输入。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    int x, y, z;
    cin >> x >> y >> z;
    cout << "x: " << x << ", y: " << y << ", z: " << z << "." << endl;
    return 0;
}
```
