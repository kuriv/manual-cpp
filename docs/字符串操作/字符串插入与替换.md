# 字符串插入与替换

执行下面的代码，在字符串中插入一个子串。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    string str("Hello World!");
    str.insert(6, "C ");
    cout << str << endl;
    str.insert(7, 6, '+');
    cout << str << endl;
    return 0;
}
```

执行下面的代码，从字符串中替换一个子串。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    string str("Hello World!");
    str.replace(6, 5, "C++");
    cout << str << endl;
    return 0;
}
```
