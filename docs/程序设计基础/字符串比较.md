# 字符串比较

执行下面的代码，将字符串从左到右每一个字符对应比较。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    string str1 = "ABC";
    string str2 = "ABE";
    cout << (str1 == str2) << endl;
    cout << (str1 != str2) << endl;
    cout << (str1 > str2) << endl;
    cout << (str1 >= str2) << endl;
    cout << (str1 < str2) << endl;
    cout << (str1 <= str2) << endl;
    return 0;
}
```
