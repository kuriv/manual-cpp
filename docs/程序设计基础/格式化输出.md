# 格式化输出

执行下面的代码，给一个浮点数指定总的显示位数。

```cpp
#include <iostream>
#include <iomanip>

using namespace std;

int main(int argc, char const *argv[])
{
    float f = 12.3456;
    cout << setprecision(3) << f << endl;
    cout << setprecision(4) << f << endl;
    cout << setprecision(5) << f << endl;
    return 0;
}
```

执行下面的代码，强制数字显示为非科学计数法的形式。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    cout << 232123434.357 << endl;
    cout << fixed << 232123434.357 << endl;
    return 0;
}
```

可以结合以上操作来强制显示小数点后的位数。

```cpp
#include <iostream>
#include <iomanip>

using namespace std;

int main(int argc, char const *argv[])
{
    double d1 = 345.4567;
    double d2 = 78676.887234;
    cout << fixed << setprecision(2) << d1 << endl;
    cout << fixed << setprecision(3) << d2 << endl;
    return 0;
}
```

也可以使用下面的方法来强制浮点数显示小数点。

```cpp
#include <iostream>
#include <iomanip>

using namespace std;

int main(int argc, char const *argv[])
{
    cout << 1.23 << endl;
    cout << setprecision(6) << 1.23 << endl;
    cout << showpoint << 1.23 << endl;
    cout << showpoint << 123.0 << endl;
    return 0;
}
```

执行下面的代码，指定输出的最小列数。

```cpp
#include <iostream>
#include <iomanip>

using namespace std;

int main(int argc, char const *argv[])
{
    cout << setw(8) << "C" << setw(6) << 233 << endl;
    cout << setw(8) << "C++" << setw(6) << 6.66 << endl;
    cout << setw(8) << "Hello" << setw(6) << 100 << endl;
    return 0;
}
```

执行下面的代码，指定输出的最小列数，并分别设置输出为右对齐和左对齐。

```cpp
#include <iostream>
#include <iomanip>

using namespace std;

int main(int argc, char const *argv[])
{
    cout << right;
    cout << setw(8) << 233 << endl;
    cout << setw(8) << 666.666 << endl;
    cout << left;
    cout << setw(8) << 233 << endl;
    cout << setw(8) << 666.666 << endl;
    return 0;
}
```
