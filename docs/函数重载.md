# 函数重载

函数重载可以用同样的名字命名函数，只要函数的参数不同。

```cpp
#include <iostream>

using namespace std;

int max(int x, int y)
{
    return x > y ? x : y;
}

double max(double x, double y)
{
    return x > y ? x : y;
}

double max(double x, double y, double z)
{
    return max(max(x, y), z);
}

int main(int argc, char const *argv[])
{
    cout << max(3, 4) << endl;
    cout << max(3.0, 5.4) << endl;
    cout << max(3.0, 5.4, 10.14) << endl;
    return 0;
}
```
