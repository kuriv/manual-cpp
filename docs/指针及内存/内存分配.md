# 内存分配

C++ 支持动态分配持久的内存空间。

```cpp
#include <iostream>

using namespace std;

int main(int argc, char const *argv[])
{
    int *p1 = new int(233);
    cout << *p1 << endl;
    delete p1;
    int *p2 = new int[6];
    *(p2 + 2) = 3;
    cout << *(p2 + 2) << endl;
    delete [] p2;
    return 0;
}
```
